Kaazing iOS HTML5 Demo

Kaazing WebSocket Gateway is a platform for building web applications that require high-performance, full-duplex communication between the browser and server. The demonstration on this page show how iOS applications can utilize WebSocket. Source code for this demo is located in the KAAZING_HOME/demo/ios/src/core directory of this release.
